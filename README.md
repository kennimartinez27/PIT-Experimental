# Sistema de Encuestas Turísticas PIT - MVP

## Objetivo del MVP
Sistema web PWA para captura digital de encuestas turísticas, 
eliminando el proceso manual en papel.

## Funcionalidades Incluidas en MVP:
✅ Login de usuarios (Admin, Encuestador)
✅ Crear/Editar encuestas con tipos de pregunta:
   - Texto corto
   - Texto largo
   - Opción múltiple
   - Escala numérica (1-5)
✅ Capturar respuestas de encuestas
✅ Ver listado de encuestas realizadas
✅ Dashboard básico con estadísticas simples

## Funcionalidades FUERA del MVP:
❌ Modo offline (Fase 2)
❌ Geolocalización (Fase 3)
❌ Generación de PDF (Fase 4)
❌ Blockchain (Futuro)
❌ Chatbot (Descartado)
